# ATM Machine Java OOPs Project

A simple ATM Machine simulation built in Java using Object-Oriented Programming concepts such as classes, objects, methods, encapsulation, and constructors.

## Features
- User PIN verification
- Check account balance
- Withdraw money
- Deposit money
- Menu-driven program
- Recursion-based re-prompting

## Technologies Used
- Java
- OOPs (Classes and Objects)
- Scanner for user input

## How to Run
1. Clone the repository
2. Open the project in IntelliJ / VS Code / Eclipse
3. Run the `ATMMachine.java` file

## Output Example
- Enter PIN  
- Withdraw/Deposit/Check Balance  
- Exit
